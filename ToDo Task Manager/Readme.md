# ✅ To Do Task Manager in C -

A simple terminal-based to-do list manager built using C. You can add, view, and delete tasks, all saved in a file.

## Features: 
- Add new tasks.
- View the existing tasks.
- Deleting tasks.

## Concepts Used:
1. Loops
2. File Handling
3. Functions
4. Conditional Statements
5. Switch Case

## 🛠️ How to Execute:
```bash
gcc todo.c -o todo
./todo
